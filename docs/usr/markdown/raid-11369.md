Delete a RAID

```bash
maas $PROFILE raid delete [--help] [-d] [-k] system_id id

```

#### **Positional arguments**
| Argument | Effect |
|---|---|
| system_id | The system ID of the machine/device (e.g., `abc123`) |
| id | The ID of the resource (e.g., `1`, `abc123`) |


#### **Command-line options**
| Option | Effect |
|---|---|
| --help, -h | Show this help message and exit. |
| -d, --debug | Display more information about API responses. |
| -k, --insecure | Disable SSL certificate check. |


#### **Keywords**
Delete a RAID with the given id on a machine with the<br>given system_id.

Update a RAID

```bash
maas $PROFILE raid update [--help] [-d] [-k] system_id id [data ...]

```

#### **Positional arguments**
| Argument | Effect |
|---|---|
| system_id | The system ID of the machine/device (e.g., `abc123`) |
| id | The ID of the resource (e.g., `1`, `abc123`) |


#### **Command-line options**
| Option | Effect |
|---|---|
| --help, -h | Show this help message and exit. |
| -d, --debug | Display more information about API responses. |
| -k, --insecure | Disable SSL certificate check. |


#### **Keywords**
Update a RAID with the given id on a machine with the<br>given system_id.

##### Keyword "name"
Optional String. Name of the RAID.
##### Keyword "uuid"
Optional String. UUID of the RAID.
##### Keyword "add_block_devices"
Optional String. Block devices to<br>add to the RAID.
##### Keyword "remove_block_devices"
Optional String. Block devices<br>to remove from the RAID.
##### Keyword "add_spare_devices"
Optional String. Spare block<br>devices to add to the RAID.
##### Keyword "remove_spare_devices"
Optional String. Spare block<br>devices to remove from the RAID.
##### Keyword "add_partitions"
Optional String. Partitions to add to<br>the RAID.
##### Keyword "remove_partitions"
Optional String. Partitions to<br>remove from the RAID.
##### Keyword "add_spare_partitions"
Optional String. Spare<br>partitions to add to the RAID.
##### Keyword "remove_spare_partitions"
Optional String. Spare<br>partitions to remove from the RAID.


Note: This command accepts JSON.
