Upload chunk of boot resource file.

```bash
maas $PROFILE boot-resource-file-upload update [--help] [-d] [-k] resource_id id

```

#### **Positional arguments**
| Argument | Effect |
|---|---|
| resource_id | The resource_id parameter |
| id | The ID of the resource (e.g., `1`, `abc123`) |


#### **Command-line options**
| Option | Effect |
|---|---|
| --help, -h | Show this help message and exit. |
| -d, --debug | Display more information about API responses. |
| -k, --insecure | Disable SSL certificate check. |


#### **Keywords**
Uploads a chunk of boot resource file
