Delete a VLAN

```bash
maas $PROFILE vlan delete [--help] [-d] [-k] fabric_id vid

```

#### **Positional arguments**
| Argument | Effect |
|---|---|
| fabric_id | The fabric_id parameter |
| vid | The vid parameter |


#### **Command-line options**
| Option | Effect |
|---|---|
| --help, -h | Show this help message and exit. |
| -d, --debug | Display more information about API responses. |
| -k, --insecure | Disable SSL certificate check. |


#### **Keywords**
Delete VLAN on a given fabric.

Update VLAN

```bash
maas $PROFILE vlan update [--help] [-d] [-k] fabric_id vid [data ...]

```

#### **Positional arguments**
| Argument | Effect |
|---|---|
| fabric_id | The fabric_id parameter |
| vid | The vid parameter |


#### **Command-line options**
| Option | Effect |
|---|---|
| --help, -h | Show this help message and exit. |
| -d, --debug | Display more information about API responses. |
| -k, --insecure | Disable SSL certificate check. |


#### **Keywords**
Updates a given VLAN.

##### Keyword "name"
Optional String. Name of the VLAN.
##### Keyword "description"
Optional String. Description of the VLAN.
##### Keyword "mtu"
Optional Int. The MTU to use on the VLAN.
##### Keyword "dhcp_on"
Optional Boolean. Whether or not DHCP should<br>be managed on the VLAN.
##### Keyword "primary_rack"
Optional String. The primary rack<br>controller managing the VLAN (system_id).
##### Keyword "secondary_rack"
Optional String. The secondary rack<br>controller managing the VLAN (system_id).
##### Keyword "relay_vlan"
Optional Int. Relay VLAN ID. Only set when<br>this VLAN will be using a DHCP relay to forward DHCP requests to<br>another VLAN that MAAS is managing. MAAS will not run the DHCP relay<br>itself, it must be configured to proxy reqests to the primary and/or<br>secondary rack controller interfaces for the VLAN specified in this<br>field.
##### Keyword "space"
Optional String. The space this VLAN should be<br>placed in. Passing in an empty string (or the string 'undefined') will<br>cause the VLAN to be placed in the 'undefined' space.


Note: This command accepts JSON.
