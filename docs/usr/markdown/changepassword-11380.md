Change a MAAS user&#39;s password.

```bash
maas changepassword [-h] [--database {default}] [username]

```


#### **Command-line options**
| Option | Effect |
|---|---|
| -h, --help | show this help message and exit. |
| --database {default} | Specifies the database to use. Default is “default”. |




#### **Positional arguments**
| Argument | Effect |
|---|---|
| username | Username to change password for; by default, it&#39;s the current username. |
